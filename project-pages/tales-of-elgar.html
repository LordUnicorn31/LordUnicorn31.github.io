Tales of Elgar
Tales of Elgar is a 2D RPG created in unity. It is a project designed and programmed by me. The game is in a vertical slice state.
Project Details / Background
For this project I created several design documents, including a GDD, a worldbuilding document and a map. I am currently developing the game having already implemented systems such as Inventory, Quests and Dialogue.
As game designer I created the GDD, the worldbuilding document and a document including all the playable characters and main enemies (inclusing their personalities, physical description and main traits). I also did a battle simulator in order to balance enemies and allies depending on their strength/weakness.
I made a first pre-alpha demo using rpg maker containing a few main quests and combats.
I am currently working on the production of the game using Unity Engine. Dialog, Quest and inventory systems have been created. The game is in a vertical slice state.
